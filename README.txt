README: Botrytis_ImageAnalysis
This repository contains scripts for general Botrytis image analysis, 
as well as instructions to lead you through the full protocol.

~/notes contains instructions and GitHub installation instructions
~/scripts contains image analysis scripts
~/photos contains sample images for practice using the scripts

create a new subdirectory ~/images and do NOT stage/commit it if you want to store
your own image files locally
you can also add the ~/images subdirectory to your .gitignore file
